06 — Gallery Formats

Three movements in the gallery section. Each has a distinct compositional logic — no two read the same.

.gallery__plate — full-bleed plate

Single cinematic image at 86vh. Caption in eyebrow small-caps, bottom-left, cream/55. Clip-path mask reveal on enter.

.gallery__pair — asymmetric pair

4fr portrait (3:4) + 7fr landscape (16:10). The wide image is offset translateY(8%) to break the baseline — prevents a static lockstep grid.

.gallery__pin — pinned horizontal reel

On desktop: ScrollTrigger pins the container and scrubs translateX across 5 panels (scrub: 0.6 — page settles, doesn't snap). On mobile (<900px) becomes a native horizontal scroller with scroll-snap. Preview below shows the mobile fallback.

07 — Eyebrow / Labels

Single class, two colour states. Used for section markers, metadata tags, image captions, nav links — anywhere the type must label without shouting.

.eyebrow + .eyebrow--solid

Default is --ink-55 (muted). --solid modifier lifts to full ink. On [data-theme="ink"] contexts the variables swap automatically to cream equivalents.

08 — Cursor

12px disc that lerps to pointer at factor 0.12. Scales to 64px over media zones. mix-blend-mode: difference inverts colour against the image.

#cursor — base state

Hidden on touch/coarse pointer, hidden under prefers-reduced-motion. Position driven by gsap.quickSetter so scale tween doesn't conflict.

09 — Reveal Patterns

GSAP entrance choreography. All fires once on ScrollTrigger enter, controlled by a single gsap.context().

Image mask reveal

clip-path: inset(100% 0 0 0) → inset(0%) over 0.9–1.0s expo.out, paired with scale 1.06 → 1 over 1.6s power3.out. Applied via [data-reveal="mask"].

Char stagger (hero title)

yPercent: 110 → 0 per character, stagger: 0.025s. Fired at page load, not scroll. Total ~1.2s for "En Coulisses" (12 chars).

Word stagger (synopsis / method)

yPercent: 100, opacity: 0 → 0, 1 per word, stagger: 0.03–0.04s. ScrollTrigger fires at 78–82% viewport. Handled by splitText(el, 'words').

Reduced motion: All clip-path masks, parallax scrubs, Lenis smooth-scroll, and the pinned gallery are stripped. Replaced with 0.3s opacity fades. Content is always accessible and visible — the choreography is an enhancement, not a dependency.
